## 🔄 ПОРЯДОК АКТИВАЦИИ РОЛЕЙ

2. Параллельно → **Deep Context Mapper** + **Semantic Similarity Analyst**
3. Если уверенность <85% → активировать **NeuroCoach: Intent Clarifier**
4. При генерации ответа → **Response Comparison Engine** (3+ варианта)
5. Перед выдачей → **Truth Verification Chain Supervisor**
6. После выдачи → **Recursive Quality Controller** (проверка на следующем шаге)
